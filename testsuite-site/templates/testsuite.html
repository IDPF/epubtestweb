Testsuite

A freely available collection of material created specifically for the purpose of testing reading systems is provided here. The source from which these downloadable publications are built can be found on github. Prebuilt versions are available at the bottom of this page.

Instructions for evaluators

If you are interested in becoming an evaluator, please contact info@bisg.org. Once you are a registered user of this site (it's free!), your results will be stored in the central database and available for your retrieval but will not be published unless you would like to contribute to the BISG EPUB3 Support Grid (the public results available on the front page). Contact info@bisg.org for more information on the publication process.

If you are not interested in becoming an evaluator, you are still encouraged to download the testsuite documents below to test your reading system.

The following instructions apply to logged-in users.

  • Download the zip archive from the list below.
  • Log in.
  • Go to Manage.
  • Choose Add Reading System.
  • Enter the relevant details for your reading system.
  • Click Save.
  • From the Actions column, choose Evaluate.
  • Open your reading system and load each publication. You'll see a series of tests.
  • Follow the instructions for each test.
  • Back at epubtest.org, record the results for each test, using its ID to match the test in the book with the test on the website form. If the test passes, record the result as Supported. Otherwise, put down Not Supported.
  • Note that you can save an incomplete evaluation form and return to finish it later. Also note that evaluation forms autosave every 30 seconds.
  • When you are done with all of the tests, contact info@bisg.org to publish your results.
  • From time to time, the testsuite grows. You are encouraged to check back regularly for alerts that your evaluation requires updating.
