Testsuite

A freely available collection of material created specifically for the purpose of testing reading systems is provided here. The source from which these downloadable publications are built can be found on github.

Instructions for testers

Add your reading system

  • Log in.
  • Go to Manage.
  • Choose Add Reading System.
  • Enter the relevant details for your reading system.
  • Click Save.
  • From the Actions column, choose Evaluate.
  • Follow the instructions below. Note that you can save an incomplete evaluation form and return to finish it later. Also note that evaluation forms autosave every 30 seconds.
  • Reading Systems and other individuals may use the test suite to test privately; results will not be published unless you would like to contribute to the BISG EPUB3 Support Grid. Contact info@bisg.org for more information on the publication process.

Using the testsuite documents

  • Download all the files below.
  • Open each publication in your reading system. You'll see a series of tests.
  • Follow the instructions for each test.
  • Record the result using the test ID (something like style-020) to identify the test.
  • If you are a logged-in user of this website, use the form provided for reading system evaluations (under "Manage"). If the test passes, record the result as Supported. Otherwise, put down Not Supported.
  • If you are using a different method to record results, be sure to make note of the test ID (as mentioned above).
