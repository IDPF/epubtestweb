Instructions for Evaluators

The following instructions apply to logged-in users.

  • Log in.
  • Go to Manage.
  • Choose Add Evaluation and follow the instructions on screen. Unless you are performing accessibility testing, choose EPUB 3 as your Test Suite.
  • You may first wish to add a reading system if it's not already listed on the site.
  • Start filling out the evaluation details and proceed to the first section.
  • Download the EPUBs required for each section. They are listed at the top of the relevant evaluation section.
  • Follow the instructions for each test.
  • Record the results for each test, using its ID to match the test in the book with the test on the website form. If the test passes, record the result as Supported. Otherwise, choose Not Supported.
  • Note that you can save an incomplete evaluation form and return to finish it later.
  • When you are done with all of the tests, go to your Manage page and click the link Request to publish to publish your results.
  • From time to time, the Test Suite grows. You are encouraged to check back regularly for alerts that your evaluation requires updating
