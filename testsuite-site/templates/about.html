About

This website presents a summary of Reading System performance evaluations for features in EPUB 3.0. The feature tests come from the EPUB Testsuite, a freely available collection of material created specifically for this purpose. The primary focus of the collection is on binary (pass/fail) tests. At present, the current testing process involves that evaluation and recording of results by humans (as opposed to being fully automated, since not all EPUB Reading Systems presently support JavaScript and/or network access)."
